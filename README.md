# 1.Retail Sales Analysis

## Objective
Analyze supermarket sales data to understand daily trends,
best-selling products, and customer behavior.

## Dataset
- File: `supermarket_sales.csv`
- Columns: Date, Product, Quantity, Total, CustomerID

## Tools
- Python
- pandas
- matplotlib
- Jupyter Notebook

## How to Run
1. Open `project1_retail_sales_analysis.ipynb` in Jupyter Notebook.
2. Run all cells sequentially.
3. Visualizations will show sales trends and top products.
